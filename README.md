# What's Going On?
Do you feel like the news around you are overwhelming? Do you feel like everything is happening all at once? This app allows you to see exactly what and were it's happening!
This is a simple react app which pulls RSS feeds, parses their contents for geospatial data and then places markers on a map to show you exactly what's going on, where, how much coverage there is and etc.
___
This is meant to be a simple project for learning the basics of react and for the sake of convenience
